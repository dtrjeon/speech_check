Section 01

앱 개요 및 화면 구성

전도폭발(EE) 훈련생이 복음제시 대사를 소리 내어 연습하고, 녹음된 음성과 원본 텍스트를 단어 단위로 비교하여 암기 정확도를 점검하는 도구입니다.

화면 구성
사용자 바로그인 이름·반·단계 표시, 로그아웃 버튼
① 챕터 선택암기할 장(章)을 1개 이상 선택
② 원본 텍스트선택된 챕터의 정답 대사 자동 로드
③ 음성 입력STT 또는 클로바노트로 녹음 텍스트 입력
④ 비교 분석원본과 녹음 텍스트를 단어 단위로 비교
⑤ 결과 패널일치율·누락·추가·대체 단어 색상 표시
💡 우측 상단 🟢 온라인 연결 표시로 인터넷 연결 상태를 확인할 수 있습니다.
우측 하단 🌙 버튼으로 화면 테마를 변경합니다.
Section 02

로그인 · 로그아웃

앱을 열면 로그인 화면이 표시됩니다. 관리자(2단계 교사)에게 발급받은 이름과 비밀번호를 입력해야 사용할 수 있습니다.

  • 이름 입력 후 엔터 → 비밀번호 칸으로 이동합니다.

  • 비밀번호 입력 후 로그인 버튼을 탭하거나 엔터를 누릅니다.

  • 성공하면 상단 사용자 바에 이름 · 반 · 단계가 표시됩니다.

✅ 로그인 정보는 브라우저 세션 동안 유지됩니다. 브라우저를 완전히 닫으면 자동 로그아웃됩니다.
⚠ 비밀번호는 2단계 교사에게 문의해주세요.

로그아웃

상단 사용자 바 오른쪽 로그아웃 버튼을 탭하면 즉시 로그아웃됩니다.

Section 04

원본 텍스트 확인

챕터를 선택하면 원본 텍스트 카드에 정답 대사가 자동으로 채워집니다.

카드 조작 버튼
접기/펼치기헤더 오른쪽 버튼으로 텍스트 영역을 숨기거나 표시
전체보기↗ 아이콘 → 전체화면 모달로 크게 표시
글자 크기가% 버튼 반복 탭 → 100%~150% 5%씩 증가
✅ 성경 구절([SKIP]...[/SKIP])은 비교 분석 시 자동 제외되며, 삭제된 자리에는 공백 1칸이 유지됩니다.
Section 05

음성 입력 — 실시간 STT

⚠ 아이폰(iOS)은 STT를 지원하지 않습니다. 6절 클로바노트 방식을 사용하세요.

  • 우측 상단 ⚙ STT 설정 → 🎙️ 실시간 STT 선택합니다.

  • 🎤 마이크 권한 허용 버튼을 눌러 마이크 접근을 허용합니다.

  • ● 녹음 시작 버튼을 누르고 복음 대사를 또렷하게 읽습니다. 버튼이 빨간 맥박 애니메이션으로 변하면 녹음 중입니다.

  • 말하는 내용이 실시간으로 텍스트 변환되어 미리보기에 표시됩니다.

  • 읽기가 끝나면 ■ 녹음 중지를 누릅니다.

  • 저장 버튼으로 녹음 텍스트를 확정합니다.

💡 저장 방식: 둘 다(기본) / 덮어쓰기 / 추가 선택 가능
Section 06

음성 입력 — 클로바노트

📱 네이버 클로바노트 앱(무료)을 이용하면 아이폰에서도 고품질 음성-텍스트 변환이 가능합니다.
방법 A — 파일 저장 후 불러오기

  • 클로바노트 앱 → 녹음 → 변환 완료 → 공유 → .txt 저장

  • 앱 클로바노트 탭 → 📂 파일 불러오기

  • 녹음 텍스트로 적용하기

방법 B — 직접 붙여넣기

  • 클로바노트 앱 → 녹음 완료 → 전체 선택 → 복사

  • 이 앱 → 클로바노트 탭 텍스트 칸 → 길게 탭 → 붙여넣기

  • 녹음 텍스트로 적용하기

Section 07

비교 분석 실행

  • 챕터 선택 → 원본 텍스트 확인 → 녹음 텍스트 입력을 완료합니다.

  • ＋ 비교 분석 버튼을 누릅니다.

  • 화면 하단에 결과 패널이 펼쳐집니다.

Section 08

결과 화면 보는 법

일치율 점수

결과 패널 상단에 % 일치율과 등급이 표시됩니다. 4개 통계 칩으로 전체·일치·누락·추가 단어 수를 확인하세요.

색상 범례

단어일치 — 원본과 정확히 일치
단어누락 — 원본에 있지만 말하지 않음 (취소선)
단어추가 — 원본에 없는데 말한 단어
단어대체 — 비슷하지만 다른 단어

2단 비교 보기

원본(좌)과 내 발화(우)를 나란히 표시하여 오류 위치를 한눈에 파악할 수 있습니다.

Section 11

홈화면 설치 (PWA)

앱을 스마트폰 홈화면에 설치하면 일반 앱처럼 아이콘으로 실행할 수 있습니다.

🍎 아이폰 — Safari

  • Safari에서 앱 주소를 엽니다.

  • 하단 가운데 공유 버튼 (□↑)을 탭합니다.

  • "홈 화면에 추가"를 선택합니다.

  • 이름 확인 후 추가를 탭합니다.

🤖 안드로이드 갤럭시 — Chrome

  • Chrome에서 앱 주소를 엽니다.

  • 우측 상단 ⋮ 메뉴를 탭합니다.

  • "홈 화면에 추가" 또는 "앱 설치"를 선택합니다.

  • 추가를 탭합니다.

Section 13

자주 묻는 질문 (FAQ)

Q. 아이폰에서 녹음이 안 됩니다

iOS Safari는 Web Speech API를 지원하지 않습니다. 클로바노트 탭을 이용하세요.

Q. 비밀번호를 모릅니다

2단계 교사(관리자)에게 문의하세요. 관리자는 Google Sheets user_list 시트에서 직접 변경할 수 있습니다.

Q. 여러 챕터를 한 번에 점검하려면?

챕터 카드를 순서대로 탭하여 다중 선택하면 원본 텍스트가 연결됩니다.

Q. 성경 구절이 틀렸다고 표시됩니다

앱 원본 텍스트의 성경 구절은 [SKIP] 태그로 자동 제외됩니다. 직접 입력한 텍스트에는 동일하게 태그를 추가하거나 해당 구절을 삭제하세요.

Q. 분석 결과를 파일로 저장하려면?

프린트 출력 시 브라우저 인쇄 대화상자에서 "PDF로 저장"을 선택하면 됩니다.

Q. 암기개요 설정이 초기화됩니다

암기개요 변경 후 반드시 저장 버튼을 눌러야 브라우저에 보존됩니다.

전체 사용 흐름

  • 로그인 (이름 + 비밀번호)
  • 챕터 선택 (1개 이상)
  • 원본 텍스트 확인 (자동 로드)
  • 녹음 방식 선택 (STT / 클로바노트)
  • 복음 대사 낭독 및 텍스트 저장
  • 비교 분석 버튼 실행
  • 색상 강조로 오류 확인
  • (필요 시) 미리보기 · 프린트 출력
  • 사용 종료 시 로그아웃
